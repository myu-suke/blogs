## toLocaleString

金額のフォーマット時に仕様

`Number.prototype.toLocaleString()`

単位の付与
```
number.toLocaleString('ja-JP', { style: 'currency', currency: 'JPY' })
```

小数点以下の丸め
```
number.toLocaleString('ja-JP', { maximumFractionDigits: 2 })
```
